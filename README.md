# Zeglam Dashboard — Painel IA

Dashboard de atendimento IA para a Zeglam Joias. MVP visual para apresentação de onboarding.

## Como rodar

```bash
cd zeglam-dashboard
npm install
npm run dev
```

Abra http://localhost:3000

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- Zustand (estado global)
- React Router DOM (rotas)
- Framer Motion (animações)
- Lucide React (ícones)
- date-fns (datas pt-BR)
